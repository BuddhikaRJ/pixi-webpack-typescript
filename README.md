## How to Use

1. Install NPM packages
   ```sh
   npm install
   ```
2. play game
   ```js
   npm run play
   ```
3. build game
   ```js
   npm run build
   ```
## Configuration
   * each class has configurable variables at the beginning of the class.
   * gameConfig.ts can be used to change the renderer dimensions, add assets and texture coordinate references.
